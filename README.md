## 🛠️ Getting Started

To run this project locally:
**Clone the repository**
   ```bash
   git clone https://github.com/aryapras22/simple-chat-app.git
   cd simple-chat-app
   npm install
   npm run dev
   server run on : http://localhost:3000
   ```
